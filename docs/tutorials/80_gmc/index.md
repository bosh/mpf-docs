# MPF Tutorial - 0.80 with Godot Media Controller (GMC)

Let's learn by example! Updated for 2026 with the full release
of the Godot-integrated Media Controller, MPF-GMC!

This tutorial will walk you through using MPF to create a basic pinball
machine config as well as very simple starting screens in Godot.

You don't need to have any physical pinball machine hardware to complete the tutorial -
MPF can be run virtually before you ever have to buy anything, and it is flexible
enough that you can choose from a variety of different hardware options when it
comes time to build.

## Contents

This tutorial includes:

* Configuring switches, coils, flippers, sling shots, and your trough.
* Starting and playing a complete game with multiple players.
* Setting up attract mode light and display shows.
* Basic scoring and defining shots and lights.
* Using the display to show what's happening and the score.
* Setting up a "base" game mode.

The tutorial's steps each build off the previous step, so it is important
to follow it in order. After completing the tutorial, you can browse the
rest of the documentation and read "How To" guides for everything else you
want to add in.

The [legacy tutorial](../legacy_mc/index.md) has an accompanying video walk-through,
seen below. Much of the core pinball configuration is still the same, though anything
related to media (video and sound) will be very different now with the upgrade to MPF 0.80.
