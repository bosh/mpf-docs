## Latest Versions

If you are creating a new game and are installing all of the MPF tools from scratch, you should start with the most up-to-date versions of everything you can. This will ensure you have the latest features and bug fixes, and are the easiest for the community to support.
The most updated full installation (confirmed working on Windows and Linux) as of April 2026 is:

* Python 3.14.3 ([python.org](https://www.python.org/downloads/release/python-3143/))
* MPF 0.80.0 (via pip ([PyPi](https://pypi.org/project/mpf/0.80.0/)))
* MPF-GMC 0.1.6 (via [Godot Asset Library](https://godotengine.org/asset-library/asset/3125))
* Godot Editor 4.6 ([4.6.2](https://godotengine.org/download/))
* MPF Monitor 0.57.2 (via pip ([Pypi](https://pypi.org/project/mpf-monitor/0.57.2/)))
